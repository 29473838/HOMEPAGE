# ShowkerTMS - 소리가 담긴 타임머신 (Flask 버전)

## 구성
- Flask + SQLite 기반
- 소개 / 링크 / 공지 / 비즈니스 / 문의 페이지
- 로그인 / 회원가입
- 공지 작성/수정/삭제 (관리자만)
- 댓글 작성 (로그인한 유저만)
- 벚꽃 애니메이션 + 배경 음악 + 네온 타이틀 디자인

## 설치 방법

```bash
pip install flask werkzeug
```

(가상 환경 사용을 추천합니다.)

## 실행

```bash
python app.py
```

브라우저에서 다음 주소로 접속합니다.

http://127.0.0.1:5000/

## 기본 관리자 계정

- 아이디: admin
- 비밀번호: admin123

관리자로 로그인하면:
- /notice 페이지에서 "새 공지 작성" 버튼이 보입니다.
- 공지 상세 페이지에서 수정/삭제가 가능합니다.

일반 유저는 회원가입 후:
- 공지 읽기 + 댓글 작성만 가능합니다.

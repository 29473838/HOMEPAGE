오픈소스 라이선스 안내

ShowkerTMS는 여러 오픈소스 소프트웨어의 도움을 받아 제작되었습니다. 각 프로젝트와 라이선스 정보를 아래에 안내합니다. 상세한 내용은 해당 프로젝트의 공식 저장소 또는 홈페이지를 참고바랍니다.

1. TOAST UI Editor

  • 홈페이지 / 저장소: https://github.com/nhn/tui.editor
  • 라이선스: MIT License
  • 주요 사용처: 공지사항 작성/수정 에디터, 서식 지정 기능 제공

2. Flask

  • 저장소: https://github.com/pallets/flask
  • 라이선스: BSD-3-Clause

3. SQLAlchemy

  • 저장소: https://github.com/sqlalchemy/sqlalchemy
  • 라이선스: MIT License

4. 기타 사용 라이브러리

이 외에도 Flask 생태계의 다양한 라이브러리(Flask-Login, Flask-Mail 등)를 사용하고 있습니다. 각 라이브러리의 세부 라이선스는 해당 프로젝트 페이지에서 확인하실 수 있습니다.

※ 이 페이지는 서비스에서 사용 중인 오픈소스 구성에 따라서 수정·보완될 수 있습니다.
